Welcome To The Simon Game!

How To Play:

  1. Pay attention to the sequence of colors that the game presents to you. It will start with a single color and then gradually add more colors as you progress.

  2. Click the buttons in the same order as the game shows you. Each click should correspond to one color in the sequence.

  3. If you get the sequence correct (Score will go up by +1) all boxes will flash green three times and it will add a new color to the sequence for you to follow.

  4. If you get the sequence wrong all boxes will flash red three times and your score will be reset back to 0 and you need to restart the game

  5. To start game press Start Game!
